# Faerun
Faerun game in graphic (Java swing)

This is a game called "The battle of Faerun" which is inspired by creatures of the Tolkien World. There are multiple modes :
- Singleplayer
- Server
- Client

In the game you're in charge of a castle were you can create units of differents types and once they're trained they'll go on the battle field to face the opponent.
